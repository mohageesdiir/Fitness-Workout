About This Website Fitness Workout Planner

The Fitness Workout Planner is a complete web application designed to help users track their BMI, browse target-muscle exercises, and organize weekly workout schedules with ease. This project combines HTML, CSS, and JavaScript to deliver a clean UI, smooth user interactions, and real-time fitness tracking features.

Main Features

  1. BMI Calculator
  2. Browse Exercises
  3. Weekly Workout Scheduler
  4. Calendar System
  5. Workout Timer Page
